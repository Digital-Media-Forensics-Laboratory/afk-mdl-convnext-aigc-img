# afk-mdl-convnext-aigc-img

基于 ConvNeXT 系列模型的AIGC图像鉴别模型

语言:
- [English](README.md)
- [中文](README_zh.md)

本模型用于AIGC图像鉴别，支持检测常见AIGC模型如Stable Diffusion，也支持对应用软件生成的图像（如豆包AI）进行检测。此外，模型支持JPEG压缩图像的检测。

本模型处于验证阶段，权重将会在未来迭代。
